# WiFi Password Viewer (Windows)

A simple Python project that shows saved Wi-Fi profiles and their stored passwords on your own Windows computer using built-in `netsh` commands.

## Features

- Show all saved Wi-Fi profiles
- Display saved passwords
- Beginner friendly code
- Uses only Python standard library
- No external packages needed

## Requirements

- Windows
- Python 3.x

## How to Run

1. Install Python
2. Save the script as `password.py`
3. Open Command Prompt or PowerShell
4. Run:

```bash
python password.py
